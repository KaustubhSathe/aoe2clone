# Age of Empires II Clone - Implementation Roadmap

This document outlines a proposed roadmap for the next logical steps in developing the AoE2 clone. The current game features a working isometric rendering engine, basic fog of war, minimap, basic UI, villager movement with collision, and Town Center training queues.

The next steps focus on turning this technical foundation into a playable RTS game loop.

## Proposed Milestones

### Milestone 1: Town Center 
- [x] Add town center info in Selection Info column, add Health title infront of the health bar
- [x] Add Capacity title below it which indicates garrison capacity of the town center. It should be Capacity: 0/15 when ungarrisoned and increases when villagers are garrisoned in the town center.
- [x] Below it Add Attack title , Attack: 5 
- [x] Below it Add Range title, Range: 6
- [x] Also implement gather point for town center. I.e when a villager is produced from town center it should gather at the gather point.
- [x] Town center garrison feature, maximum 15 villagers can garrison in town center. When garrisoned, the villager should not be able to move or perform any other action. Also if gather point of town center is set to town center itself then the new villager created should get garrisoned in the town center.
- [x] Use flat building asset for town center

### Milestone 2: Grid-Based Pathfinding (A*)
- [x] Villagers currently move in a straight line and stop if they hit a blocked tile.
        *   **A* Algorithm:** Implement standard A* pathfinding on the `GRID_SIZE x GRID_SIZE` map.
        *   **Path Following:** Update the villager movement logic to follow a list of tile waypoints generated by A* rather than just moving directly to `targetPosition`.
        *   **Dynamic Obstacles:** Ensure the pathfinder respects dynamic changes (like trees being chopped down, or new buildings being placed).
        * Implement this for all selected units. Group path finding.
